== Description ==

Makes post thumbnails link to post with post title as image title


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Extract `nnd-link-post-thumbnails-to-post` from `nnd-link-post-thumbnails-to-post.[version #].zip`
2. Upload `nnd-link-post-thumbnails-to-post` and contents to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
